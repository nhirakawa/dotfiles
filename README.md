## How it works
Dotfiles will be symlinked to `~` according to `conf/index.json`.

## Usage
```
usage: Install dotfiles [-h] [-b BACKUP]

optional arguments:
  -h, --help            show this help message and exit
  -b BACKUP, --backup BACKUP
```
